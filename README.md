# BroadcastDiscordBot

Broadcast Discord Bot Sent To All User With A Single Command

# Requirment

1- node.js v22 Or Higher

2- install NPM 

3-install discord.js Node modules

4-install .env ("dotenv") Node modules

# Installing

```
npm install
```
```
npm install discord.js
```
```
npm install dotenv
```
